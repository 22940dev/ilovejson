# I ❤️ JSON

## Prerequisites
- NodeJS v13.x
- Visual Studio Code

## Getting Started

Installing Dependencies
```bash
npm install
# or
yarn install
```

Run Server

```bash
npm run dev
# or
yarn dev
```
